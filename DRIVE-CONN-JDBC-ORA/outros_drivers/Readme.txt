ojdbc11-full.tar.gz - JDBC Thin Driver and Companion JARS
========================================================
This TAR archive (ojdbc11-full.tar.gz) contains the 23.4.0.24.05 release of the Oracle JDBC Thin driver(ojdbc11.jar), the Universal Connection Pool (ucp11.jar) and other companion JARs grouped by category. 

(1) ojdbc11.jar (7170864 bytes) - 
(SHA1 Checksum: 239c6f65f74ebdf08bf0da234815fc757a13224c)
Oracle JDBC Driver compatible with JDK8, JDK11, JDK12, JDK13, JDK14, and JDK15.
 
(2) ucp11.jar (1533329 bytes) - (SHA1 Checksum: eacebf2a63a742c3dffaf169e0f4b75bb66c828e)
Universal Connection Pool classes to be used with ojdbc11.jar -- for performance, scalability, high availability, sharded and multitenant databases.

(3) rsi.jar (260434 bytes) - (SHA1 Checksum: aef073fffc280995e8ae720d16e6296da691b38c)
Reactive Streams Ingestion (RSI) 

======================
Security Related JARs
======================
Java applications require some additional jars to use Oracle Wallets. 
You need to use all the three jars while using Oracle Wallets. 

(4) oraclepki.jar (501935 bytes) - (SHA1 Checksum: 30746467bb87c82135b1e88581c27a607175aae5)
Additional jar required to access Oracle Wallets from Java
